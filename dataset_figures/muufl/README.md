# MUUFL Gulfport Dataset Visualization

## Source Data
- **Raw scene**: `MFT_Data/MUUFL_raw/MUUFLGulfport-0.1/` (University of Southern Mississippi)
- **HSI**: 325 × 220 × 64 bands (ROSIS sensor, float32) — originally 72 bands, 8 removed (first/last noisy bands)
- **LiDAR**: 325 × 220 × 2 channels (two co-registered LiDAR returns, first return elevation from each)
- **Ground Truth**: 11 classes, from `sceneLabels` in the scene .mat file (-1 = unlabeled)
- **Train/Test split**: Random 5% train / 95% test (seed=42), 2,684 train / 51,003 test pixels
- **Total labeled pixels**: 53,687 out of 71,500 (24.9% unlabeled)

## Pseudo-color Map
- **Bands**: R=40, G=20, B=10 (from original MFT paper, Fig. 5 caption)
- **Processing**: 2nd/98th percentile clipping + min-max normalization + gamma=0.85
- Shows UH Gulfport campus with buildings, trees, roads

## LiDAR Map
- Grayscale from first LiDAR channel (first return elevation) with histogram equalization

## Classes
1. Trees, 2. Grass-Pure, 3. Grass-Ground, 4. Dirt-Sand, 5. Road, 6. Water,
7. Shadow-Building, 8. Buildings, 9. Sidewalk, 10. Yellow-Curb, 11. Cloth-Panels
