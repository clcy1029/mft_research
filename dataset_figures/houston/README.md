# Houston 2013 Dataset Visualization

## Source Data
- **Raw scene**: `MFT_Data/Houston_raw/2013_DFTC/` (IEEE GRSS 2013 Data Fusion Contest)
- **HSI**: 349 × 1905 × 144 bands (CASI sensor, uint16), read from GeoTIFF with strip offset parsing
- **LiDAR**: 349 × 1905 (DSM, float32), read from GeoTIFF via PIL
- **Ground Truth**: 15 classes, parsed from ENVI ROI text files (train: 2,832 pixels, test: 12,197 pixels)
- **Train/Test split**: Disjoint fixed split from the contest (not random)

## Pseudo-color Map
- **Bands**: R=64, G=43, B=22 (from original MFT paper, Fig. 4 caption)
- **Processing**: 2nd/98th percentile clipping + min-max normalization + gamma=0.85

## LiDAR DSM Map
- Grayscale colormap with histogram equalization

## Training / Test Samples
- Colored pixels on black background, one color per class
- 15 land-cover classes
