# Trento Dataset Visualization

## Source Data
- **Raw scene**: `TrentoDateset/TrentoDataset/` (University of Trento)
- **HSI**: 166 × 600 × 63 bands (AISA Eagle sensor, float32)
- **LiDAR**: 166 × 600 (elevation, float32)
- **Ground Truth**: 6 classes, from `GT_Trento.mat` (scene-level labels)
- **Train/Test split**: Random 5% train / 95% test (seed=42), 1,510 train / 28,704 test pixels
- **Total labeled pixels**: 30,214 out of 99,600 (69.7% unlabeled background)

## Pseudo-color Map
- **Bands**: R=40, G=20, B=10 (from original MFT paper, Fig. 6 caption)
- **Processing**: 2nd/98th percentile clipping + min-max normalization + gamma=0.85
- Appears as classic false-color infrared (red = vegetation)

## LiDAR Map
- Grayscale colormap with histogram equalization

## Classes
1. Apples, 2. Buildings, 3. Ground, 4. Woods, 5. Vineyard, 6. Roads

`mft_research/visualize_datasets.py` → `visualize_trento()`
