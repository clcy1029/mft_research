# Augsburg (HSI + DSM) Dataset Visualization

## Source Data
- **Raw scene**: `MFT_Data/HS-SAR-DSM Augsburg/` (DLR, Augsburg Germany)
- **HSI**: 332 × 485 × 180 bands (HySpex sensor, uint16) — same HSI as HSI+SAR variant
- **DSM**: 332 × 485 (Digital Surface Model from DAS-EOC, float64)
- **Ground Truth**: 7 classes, from `TrainImage.mat` and `TestImage.mat` — same split as HSI+SAR
- **Train/Test split**: Pre-defined, 761 train / 77,533 test pixels

## Pseudo-color Map
- **Bands**: R=40, G=20, B=10 (from original MFT paper, Fig. 7 caption)
- **Processing**: 2nd/98th percentile clipping + min-max normalization + gamma=0.9
- Same as HSI+SAR (identical HSI source)

## DSM Map
- Terrain colormap (`plt.cm.terrain`) with histogram equalization
- Shows elevation: green = low, brown = high, white = highest

## Classes
1. Forest, 2. Residential, 3. Industrial, 4. Low Plants, 5. Allotment, 6. Commercial, 7. Water
