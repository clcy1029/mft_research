# Augsburg (HSI + SAR) Dataset Visualization

## Source Data
- **Raw scene**: `MFT_Data/HS-SAR-DSM Augsburg/` (DLR, Augsburg Germany)
- **HSI**: 332 × 485 × 180 bands (HySpex sensor, uint16)
- **SAR**: 332 × 485 × 4 bands (Sentinel-1, float64) — VV intensity, VH intensity, real component, imaginary component
- **Ground Truth**: 7 classes, from `TrainImage.mat` and `TestImage.mat`
- **Train/Test split**: Pre-defined in source files, 761 train / 77,533 test pixels
- **Total labeled pixels**: 78,294 out of 161,020 (51.4% unlabeled background)

## Pseudo-color Map
- **Bands**: R=40, G=20, B=10 (from original MFT paper, Fig. 7 caption)
- **Processing**: 2nd/98th percentile clipping + min-max normalization + gamma=0.9
- Appears as false-color (red = vegetation, cyan = urban)

## SAR Map
- RGB composite from SAR bands 0, 1, 2 with histogram equalization + gamma=0.7

## Classes
1. Forest, 2. Residential, 3. Industrial, 4. Low Plants, 5. Allotment, 6. Commercial, 7. Water
